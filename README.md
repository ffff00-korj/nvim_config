# About
My personal NeoVim configuration, written in lua. It contains:
* Lazy
* Custom bindings
* VSCode theme and others
* Lsp and lsp config
* Custom lualine
* Debugger for python
* and more...

```sh
# run NeoVim
nvim .

# type ':Lazy' abd 'I' to install plugins (it will be uploaded from github)
```
